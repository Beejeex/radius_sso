# v0.0.1 Detailed Changelog

Milestone: Project Foundation

## Added

- Added the initial Docker-first Ventus release with a multi-stage .NET 10 image, non-root runtime, Docker health check on `/health`, and Docker Compose deployment backed by PostgreSQL.
- Added the initial Ventus .NET solution layout with typed environment-variable configuration and fail-fast startup validation.
- Added unauthenticated `/health` and `/health/ready` endpoints, structured JSON logging to stdout, and OpenTelemetry tracing.
- Added Jellyfin-compatible `MediaBrowser` authorization header parsing at the external protocol boundary.
